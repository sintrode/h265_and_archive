# h265_and_archive
Convert a video file to 10-bit x265 and store it in a .xz file

# Initial Setup
Download ffmpeg and 7-Zip and place them in the folders of your choice.

Update line 18 to use the path to ffmpeg.exe and update line 20 to use the path to 7z.exe.

# Usage
    h265_and_archive.bat <input_file>
        input_file - The path to the file to process
